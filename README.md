# Venturi Flow Meterm 

## Summary 
CAD design and COMSOL imulation for a 3D printed venturi flow meter, completed as part of a group project (4 members). We designed a flow meter based around an enlarged artery and simulated blood flow by using different concentrations of corn syrup in water.  

## What did I do? 
- Completed all renditions of the CAD model for the flow meter geometry
- Debugged segments of the COMSOL simulation

## Core Principles 
- The Venturi flow meter works by constricting the fluid flow through a narrow segment, which increases the velocity of the liquid. As Bernoulli's Principle states, the increase in speed will cause a decrease in pressure of the liquid, therefore we can calculate the velocity by comparing the pressure differences.
